# The First Code

거대한 제국과 대기업이 된 플랫폼도 결국 여기서 출발했습니다.

복잡한 서비스, 글로벌 플랫폼, 수많은 사용자가 이용하는 시스템도 처음에는 작은 코드 한 줄에서 시작됩니다.

이 repository는 GitHub 사용 흐름을 익히기 위해 만든 첫 연습용 저장소입니다.

```python
print("hello, world")
```

코드 자체는 단순하지만, 목적은 분명합니다.

본격적인 데이터 분석 포트폴리오용 Python 전처리 코드와 프로젝트 문서를 GitHub에 올리기 전에, 아래 기본 과정을 직접 연습하기 위한 저장소입니다.

- repository 생성
- README 작성
- Python 파일 업로드
- 첫 commit 작성
- 공개 링크 공유
- 시크릿 모드에서 접근 확인

아직은 작은 코드 한 줄이지만, 앞으로 이 GitHub에는 데이터 수집, 전처리, 검증, 분석 흐름을 보여주는 포트폴리오 코드들이 차곡차곡 쌓일 예정입니다.

이 repository는 그 시작점입니다.
